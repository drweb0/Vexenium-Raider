# Vexenium Raider - Raid server without admin!
Vexenium Raider can send messages and blame users!
Create bot, start script, and use commands /vex (for raid), /bl (for blame user), /massping (for mass ping), /dmspam (for spam dm)


Install requirements:
```bash
pip install discord.py
````
Use commands:
```bash
/vex text:ur text delay:0.1-2.0
/bl user:@user
/massping file:users.txt amount:2
/dmspam user:@user message: ur message
````
